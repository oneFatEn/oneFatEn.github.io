机器学习的基础知识

机器学习主要分为有监督学习和无监督学习两种

有监督学习：数据集有属于自己的right answer
以买房子为例 面积与价格一一对应
比如回归以及分类（自己设定的分类代表的值）
无监督学习：将样本分为聚类

机器学习定义

A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.

E 可以看作历史数据
P 可以看作性能测量抑或是该算法的表现
T 可以看作被预测的任务

线性回归

线性回归主要解决一个最小化的问题
首先 线性回归要给出一个假设 这里以房价预测为例，每个面积对应一个价格，这样可以将所有的数据集放入二维平面中以平面直角坐标系展示。
作假设的数学公式 ：$h(x) = \theta_0 + \theta_1 x$ 即我用该假设预测非训练集中房间面积对应的价格
那么线性回归要解决的问题就是最小化训练集到该假设的距离，即使得假设最优
将距离记为$\sum_{i=1}^{m}[h_\theta(x)^\operatorname{(i)} - y^\operatorname{(i)}]^2$

运用该公式求得训练集中所有的点对于该假设的平均误差
记作代价函数：$J=\frac{1}{2m}\sum_{i=1}^{m}[h_\theta(x)^\operatorname{(i)} - y^\operatorname{(i)}]^2$
此处使用$2m$作为分母是因为m是样本的总数，2在后续的求导过程中可以被消去，为了保证公式的简洁，因此添加分母2

我们的任务就是将代价函数最小化

代价函数是以$\theta$ 为自变量的函数，此处使用的最小化方法是梯度下降法

$\theta_j := \theta_j - \alpha \frac{\partial}{\partial\theta_j}J(\theta_0,\theta_1)$
此处的的$\alpha$是控制向梯度方向下降多少的变量
同时对$\theta_0$以及$\theta_1$的更新需要同时进行 ^5930dd

$temp0:= \theta_0 - \alpha \frac{\partial}{\partial\theta_0}J(\theta_0,\theta_1)$
$temp1:= \theta_0 - \alpha \frac{\partial}{\partial\theta_0}J(\theta_0,\theta_1)$
$\theta_0 = temp0$
$\theta_1 = temp1$

采用梯度下降法时，会自动地选取越来越小的间隔，因为越向局部最优逼近，$\frac{\partial}{\partial\theta_j}J(\theta_0,\theta_1)$就越小
对于成本问题而言，其代价函数是凸函数，因此只有全局最优解，故迭代至最小时即为最优

将成本问题与梯度下降结合后
$h(x) = \theta_0 + \theta_1 x$
$J=\frac{1}{2m}\sum_{i=1}^{m}[h_\theta(x)^\operatorname{(i)} - y^\operatorname{(i)}]^2$
$\theta_j := \theta_j - \alpha \frac{\partial}{\partial\theta_j}J(\theta_0,\theta_1)$
将$h(x)$代入后得
$\frac{\partial}{\partial\theta_0}J(\theta_0,\theta_1) = \frac{1}{m}\sum_{i=1}^{m}[h_\theta(x)^\operatorname{(i)} - y^\operatorname{(i)}]$
$\frac{\partial}{\partial\theta_1}J(\theta_0,\theta_1) = \frac{1}{m}\sum_{i=1}^{m}x^\operatorname{(i)}[h_\theta(x)^\operatorname{(i)} - y^\operatorname{(i)}]$
根据这两个公式即可进行迭代进而求出最优 ^7e8d78
